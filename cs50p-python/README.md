# CS50P: Introduction to Programming with Python

This directory serves as the structured repository for my progress through Harvard’s CS50P. It contains all coursework, supplemental experimental projects, and documentation regarding my development as a programmer.

## Directory Architecture
Each weekly module is organized to ensure consistent tracking and documentation of technical growth:

* **`assignments/`**: Completed problem sets and core coursework.
* **`experiments/`**: Independent, self-directed code development used to iterate on concepts and refine logic.
* **`notes/`**: Detailed technical summaries and conceptual foundations.
* **`reflection/`**: Analysis of algorithmic challenges, problem-solving methodologies, and key learning milestones.
* **`revision/`**: Logs of code refactoring and iterative improvements made to existing projects.

## Repository Philosophy
This repository is managed with a commitment to **consistent engineering habits**. By documenting the full lifecycle of my learning—from initial conception in experiments to final implementation in assignments—I aim to build a robust foundation for future work in AI security engineering.
